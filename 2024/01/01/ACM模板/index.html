第一篇 头文件快查

本章整理了 ACM 竞赛中最常用的 16 个 C++ 头文件。

1. <algorithm> — STL通用算法

2. <bits/stdc++.h> — 全能头文件（C++17推荐）

🏆 C++ 竞赛必备！包含几乎所有常用标准库头文件。

3. <cmath> — 数学函数

4. <cstring> — 字符串处理

5. <queue> — STL队列容器

⚠️ 注意：queue 默认是 FIFO 队列，priority_queue 是优先队列（堆）。

6. <stack> — STL堆栈容器

7. <set> — STL集合容器

set / multiset：集合容器，自动排序，元素唯一/可重复。

8. <map> — STL映射容器

map / multimap：键值对容器，按键自动排序。

9. <vector> — 动态数组容器

10. <deque> — STL双端队列

双端队列：支持两端 O(1) 插入删除，随机访问。

11. <iomanip> — 参数化输入输出

12. <unordered_map> / <unordered_set> — 哈希容器

哈希表实现，查找/插入 O(1) 平均复杂度（无自动排序）。

13. <numeric> — STL数字操作

14. <bitset> — 位集容器

bitset 提供高效的位运算，支持位运算操作。
